Lancer un invité de commande depuis ce dossier

Lancez 'mongoimport --db mongonote --collection users --file users.json' pour importer les utilsateurs dans la base de données mongonote
Puis lancez 'mongoimport --db mongonote --collection notes --file notes.json' pour importer les notes dans la base de données mongonote

Vous pouvez également lancer Import.bat (inclus dans ce dossier), il est possible qu'une suppression des tables soit nécessaires, regardez les commandes en commentaires dans Import.bat
